# Website Layout Direction

## Desktop

* Hero split layout kiri teks, kanan visual
* Services grid 3x2
* Section spacing lega
* Banyak white space

## Mobile

* Hero center aligned
* Services card vertical
* Floating WhatsApp button
* CTA sticky bawah opsional

## Microcopy Style

Gunakan bahasa semi-formal:

* “Kami bantu”
* “Bisa disesuaikan”
* “Siap untuk kebutuhan bisnis Anda”
* “Modern, aman, dan scalable”

## Build Website Bintang Computer Feira

Buat website company profile modern untuk **Bintang Computer Feira — Consultan IT**.

Website harus menggunakan pendekatan **mobile-first**, dengan desain light premium modern.

## Brand Style

Gunakan palette warna:

```css
--cream-bg: #F8F1E4;
--soft-cream: #FFF9EF;
--gold: #C99A3A;
--deep-gold: #A67420;
--light-blue: #DCEFF7;
--sky-blue-soft: #BFDDEB;
--brown: #4B2E16;
--dark-brown: #2A1608;
--charcoal: #202124;
--white-soft: #FFFCF6;
```

## Visual Direction

Desain harus terasa:

* modern
* premium
* clean
* elegant
* semi-futuristic
* IT consultant style
* bukan template standar
* bukan desain kaku corporate lawas

Gunakan dekor:

* thin gold circuit lines
* soft blue cloud glow
* glassmorphism card
* rounded premium cards
* subtle gradient cream-blue
* silhouette IT engineer / server / CCTV / cloud / laptop
* soft shadow, bukan shadow kasar

## Important Mobile Requirement

Prioritaskan mobile view.

Jangan membuat desktop dulu lalu dipaksa responsive.

Masalah yang harus dihindari:

* elemen saling tindih
* gambar menutup text
* layer dekor menutup tombol
* card saling dorong
* section terlalu rapat
* navbar menutupi headline
* horizontal scroll
* fixed height yang membuat konten pecah

Mobile harus rapi di:

* 360px
* 390px
* 430px

Desktop tetap harus terlihat bagus, luas, dan premium.

Buat sebagai landing page single page dengan section:

1. Navbar

7. CTA
8. Footer

## Core Services

Tampilkan 6 produk utama:

1. IT Procurement
   Penyediaan komputer, laptop, printer, scanner, monitor, UPS, peripheral, dan kebutuhan IT kantor.

2. Network Solution
   Setup jaringan kantor, LAN, WiFi, router, firewall, server, NAS, backup system, dan private cloud.

3. CCTV System
   Pemasangan CCTV, penataan titik kamera, DVR/NVR, recording system, dan remote monitoring via smartphone.

4. PABX System
   Setup PABX, IP PBX, extension internal, call transfer, voicemail, dan sistem telepon kantor.

5. Custom Application & Refactor
   Pembuatan aplikasi custom, dashboard, reporting, operational system, HRIS, attendance, serta refactor aplikasi lama.

6. Website Development & Refactor
   Pembuatan website modern, company profile, landing page, product catalog, redesign website lama, mobile responsive, dan SEO friendly.

## Development Rules

Wajib:

* Mobile-first CSS
* Responsive grid
* No horizontal scroll
* No fixed card height
* Use clamp() for responsive typography
* Use z-index system carefully
* Decorative elements must not block content
* Images must be responsive
* Buttons minimum 48px height
* Test mobile and desktop

# Marketplace Module

Phase 3 marketplace core:

- SQLite category, product, dan product image persistence
- public versioned API untuk listing, detail, featured, category, dan search
- search, category filter, featured filter, dan sort berbasis URL
- product detail, gallery, related product, badge, dan stock status
- reusable product card, hero, campaign banner, mobile navigation, dan floating cart
- persistent member cart dengan add, update quantity, remove, dan server stock validation
- address book, shipping form, order summary, invoice, dan order history
- atomic checkout untuk mencegah overselling

Payment gateway dan payment status automation masuk Phase 5.

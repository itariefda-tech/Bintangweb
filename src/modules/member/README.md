# Member Module

Phase 2 member system:

- protected member dashboard
- profile persistence untuk identity, contact, company, address, dan bio
- GET/PUT `/api/v1/member/profile`
- same-origin dan CSRF protection untuk update
- avatar JPG/PNG/WebP maksimal 2MB dengan signature validation
- notification center persisten dan unread state
- one-hand mobile bottom navigation

Wishlist dan order data tetap menunggu phase marketplace terkait.
